# 前端交付摘要

## 当前状态

前端已与当前后端接口保持一致，支持：

- 开发模式：`npm run dev`
- 构建模式：`npm run build` 后交由 Nginx 托管

## 页面范围

### 用户侧

- 登录
- 注册
- 首页
- 活动列表
- 活动详情
- 公告详情
- 我的反馈
- 提交反馈
- 反馈详情
- 个人中心
- 个人资料页

### 管理员侧

- 活动管理
- 公告管理
- 反馈工单
- 发布活动
- 活动签到
- 时长核销
- 志愿时长统计

## 已落地的用户能力

- 登录后访问受保护页面
- 查看活动列表、详情和报名状态
- 查看首页公告、公告详情、公告附件和关联活动
- 报名活动
- 在“我的志愿足迹”中查看报名记录
- 取消尚未开始活动的报名
- 导出已核销志愿足迹 Excel
- 提交意见反馈、上传附件、追加回复和关闭反馈

## 已落地的管理员能力

- 发布活动
- 编辑活动
- 取消活动
- 结项活动
- 上传多张活动图片
- 在后台查看报名名单
- 对活动执行签到
- 对签到记录执行时长核销
- 查看志愿时长统计
- 发布、编辑、下线和删除公告
- 上传公告图片与附件，关联多个活动
- 查看反馈工单，回复、驳回、关闭反馈并调整优先级

## 当前接口约定

- 前端统一使用 `/api` 作为请求前缀
- 开发模式由 Vite 代理到 `9000`
- 部署模式由 Nginx 代理到 `9000`
- 图片读取统一走 `/api/activity/image`
- 公告图片与附件分别走 `/api/announcement/image`、`/api/announcement/attachment`
- 反馈附件走 `/api/feedback/attachments`

## 当前访问入口

- 开发模式：`http://localhost:3000`
- 本机部署模式：`http://localhost/`
- Docker 部署模式：`http://localhost:8081/`
